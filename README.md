# Hospitality Revenue Analysis

## Project Overview

This project analyzes hotel/property performance data to evaluate revenue, occupancy, booking trends, and customer behavior.

The dashboard helps stakeholders understand:
* Revenue generation
* Booking trends
* Occupancy performance
* Key hospitality KPIs

## Business Problem

Hospitality businesses need to monitor:

1. Revenue performance across properties
   
2. Occupancy and booking trends
   
3. Pricing effectiveness (ADR, RevPAR)

This dashboard provides data-driven insights to improve decision-making.

## Key Metrics (KPIs)

* Revenue: 118M
  
* Total Bookings: 7,338
  
* RevPAR: 10,640
  
* Occupancy Rate: 66%
  
* ADR (Avg Daily Rate): 16,142

## Key Insights

- High-performing properties generate majority of revenue
  
- Occupancy rate is stable around 66%
  
- Some properties show lower realization %, indicating pricing inefficiencies
  
- Revenue concentration exists in few key properties

## Business Recommendations

1. Improve pricing strategy for low-performing properties
   
2. Increase marketing for high-revenue properties
   
3. Optimize occupancy using demand forecasting
   
4. Focus on improving realization %
